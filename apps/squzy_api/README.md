## Environment variables

Bold is required

- PORT(8080) - port for listening
- **AGENT_SERVER_HOST** - host for agent server
- **MONITORING_SERVER_HOST** - host for monitoring server
- **STORAGE_SERVER_HOST** - host for storage server
